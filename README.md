# 🚀 Advance Auth System

A production-ready authentication system built with **React, Express.js, PostgreSQL, and JWT**, featuring secure authentication flows and modern best practices.

🌐 **Live Demo:** https://advance-authsystem.onrender.com

## ✨ Features

* 🔐 User Signup & Login
* 📧 Email Verification
* 🔄 JWT Authentication with Refresh Token Rotation
* 🍪 HttpOnly Cookie-Based Sessions
* 🔑 Forgot Password & Reset Password
* 🛡️ Protected Routes
* ⚡ Session Restoration on Refresh
* 📬 Brevo Email Integration

## 🛠️ Tech Stack

**Frontend:** React, Vite, Zustand, React Router, Tailwind CSS, Axios

**Backend:** Node.js, Express.js, PostgreSQL (Neon), JWT, bcrypt, Nodemailer

**Deployment:** Render

## 🚀 Run Locally

```bash
git clone https://github.com/Anmolkr007/Advance-AuthSystem.git
cd Advance-AuthSystem
npm install
npm install --prefix frontend
npm run dev
```
